--- 1.3 OSCARS: 30+ PARALEL API İSTEĞİ ---

DOSYA: context/MovieContex.js (satır ~137-162)

  tek tek API isteği atıyor. Bu Promise.all() ile paralel yapılmış
  ama yine de 30 istek = 30 API kredisi tüketimi.

  AsyncStorage'e yaz, bir kere çek, 7 gün cache'de tut:

  const fetchMoviesOscar = async () => {
    const CACHE_KEY = 'oscar_movies';
    const CACHE_TTL = 7 * 24 * 60 * 60 * 1000; // 7 gün

    const cached = await AsyncStorage.getItem(CACHE_KEY);
    if (cached) {
      const { data, timestamp } = JSON.parse(cached);
      if (Date.now() - timestamp < CACHE_TTL) {
        setMoviesOscar(data);
        setLoadingOscar(false);
        return;
      }
    }

    // İlk kez veya cache süresi dolmuşsa çek
    const results = await Promise.all(oscarIds.map(id => fetchMovieById(id)));
    const filtered = results.filter(Boolean);
    setMoviesOscar(filtered);
    AsyncStorage.setItem(CACHE_KEY, JSON.stringify({
      data: filtered,
      timestamp: Date.now()
    }));
  };

  Trending filmler 5 dakikada bir değişiyor, ama her scroll/tab

ÇÖZÜM - Basit in-memory cache:
  // Her context'in başına ekle
  const apiCacheRef = useRef({});

  const getCached = (key, ttlMs = 300000) => {
    const entry = apiCacheRef.current[key];
    if (!entry) return null;
    if (Date.now() - entry.ts > ttlMs) {
      delete apiCacheRef.current[key];
      return null;
    }
    return entry.data;
  };

  const setApiCache = (key, data) => {
    apiCacheRef.current[key] = { data, ts: Date.now() };
  };

  // Fetch fonksiyonunda kullan:
  const fetchSeriesTrends = async () => {
    const cacheKey = `trends_${selectedCategoryTrend}_${language}`;
    const cached = getCached(cacheKey, 5 * 60 * 1000);
    if (cached) {
      setSeriesTrend(cached);
      setLoadingTren(false);
      return;
    }
    // ...API isteği...
    setApiCache(cacheKey, result);
  };

--- 4.1 CHATSCREEN'DE MOUNT KORUMASI YOK ---

DOSYA: screens/ChatScreen.js (satır ~95-130)

SORUN:
  goOnline() ve goOffline() çağrılıyor ama component hızlıca
  mount/unmount olursa race condition oluşabilir.
  Kullanıcı hızlı gezinirse Firebase'e gereksiz writes gidiyor.

ÇÖZÜM:

    let isMounted = true;

    const setup = async () => {
      if (!isMounted) return;
      await goOnline();
    };

    setup();

    return () => {
      isMounted = false;
      goOffline();
    };
  }, []);

--- 4.3 TVSHOWCONTEX BAŞLANGIÇTA ÇIFT LİSTENER RİSKİ ---

DOSYA: context/TvShowContex.js (satır ~58-82)

SORUN:
  İki ayrı useEffect var:
  - Satır 58-73: mount'ta cache okuma + listener başlatma
  - Satır 76-82: auth state değişince listener başlatma

  Eğer user zaten login'se, startListener iki kez çağrılabilir.

  Senaryo:
  1. App açılıyor, cachedUserId var, startListener(cachedUid) çağrılıyor
  2. onAuthStateChanged tetikleniyor, user.uid var, startListener(uid) tekrar çağrılıyor
  3. İlk listener cleanup ediliyor (unsubRef.current() çağrılıyor) - bu doğru
  4. Ama 2 snapshot event gelebiliyor kısa sürede

ÇÖZÜM:
  İki listener başlatmayı önlemek için flag ekle:
  const initialListenerStarted = useRef(false);

  // mount effect'inde:
  if (cachedUid) {
    initialListenerStarted.current = true;
    startListener(cachedUid);
  }

  // auth effect'inde:
  if (user?.uid) {
    // Aynı UID ile zaten başlatıldıysa skip et
    if (unsubRef.current && initialListenerStarted.current) {
      initialListenerStarted.current = false;
      return;
    }
    startListener(user.uid);
  }

SORUN:

  tekrar tekrar yazılmış. Standart değil, hata-prone.

ÇÖZÜM - Merkezi cache yardımcı fonksiyonu:
  // utils/cache.js
  import AsyncStorage from '@react-native-async-storage/async-storage';

  export const getCache = async (key, ttlMs) => {
    const raw = await AsyncStorage.getItem(key);
    if (!raw) return null;
    const { data, ts } = JSON.parse(raw);
    if (ttlMs && Date.now() - ts > ttlMs) {
      AsyncStorage.removeItem(key).catch(() => {});
      return null;
    }
    return data;
  };

  export const setCache = async (key, data) => {
    await AsyncStorage.setItem(key, JSON.stringify({ data, ts: Date.now() }));
  };
